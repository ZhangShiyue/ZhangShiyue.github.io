  • Machine Translation

    RNNG

    RNNG is Chris Dyer's paper in 2016, a transition-based parsing model and its generative model can be seen as a language model. I have done some experiments based on this work since Oct. 2016. Our basic idea is to add a memory component to original structure, which memorizes wrong instances of RNNG and is supposed to work as a complement. But, our idea didn't work well and this work was halted by the end of 2016. See more info here.

    Wikipedia

    Research on Wikipedia includes article quality assessment, user reputation evaluation, collaboration patterns, and so on.

    I'm working on a project in BUPT. The main goal of this project is building up a wikipedia article quality assessment system, which is supposed to be better than or comparative to previous method and released as a online service. I have tried feature engineering combined linear regression model that got good performance. And I'm trying to introduce neural model to wikipedia article quality assessment. See more info here.

    Topic model(LDA)

    LDA(Latent Dirichlet Allocation) is a classic topic model, proposed by David M. Blei, AndrewY. Ng, Michael I. Jordan in 2003, and has been widely used during past several years. I was quite interested in this model, so I carefully reviewed its derivation process , Gibbs sampling and implemented a LDA by python .

    When I worked as an intern at YOKA.com, I participated in designing a Personalized News Recommendation system and independently implemented a 'LDA module'. See more details here.
